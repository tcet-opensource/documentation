### Steps to run the site on your local system: 
These are the steps you need to follow to get this site on your local system.

### Install Git in your computer
Follow these steps to install git in your computer.
1. Go to [https://git-scm.com/downloads](https://git-scm.com/downloads).
2. Click on Windows. Download should start.
3. Go to downloads and install the package.

```bash
git clone https://github.com/tcet-opensource/tnp-website
```

### Go to TCET Linux

1. Visit the website [TCET Linux](https://linux.tcetmumbai.in/#download)
2. Check the requirements for downloading
3. Download the 64-bit versions

:::note
A USB drive or a DVD, containing the TCET Linux installation files. This bootable media will be used to initiate the installation process and install TCET Linux on your computer.
:::

###  Linux ISO File 
1. You will need the TCET Linux ISO file, which can be downloaded from the official TCET Linux website or a trusted source. The ISO file contains the complete installation image of TCET Linux.

###  Register 
1. Adding the necessary details required to sucessfully register 

You can make changes to your respective files and changes will be shown once you have saved the file.

### Commands used to run locally 

1. To run the commands, make sure that you have installed yarn globally first.
2. All commands are run from the root of the project, from a terminal

### Here are a set of commands used to verify if its authentic:


1. For Windows

```bash
certutil -hashfile iso-name.iso sha256
```
2. For Mac 

```bash
shasum -a 256 iso-name.iso
```

3. For Linux

```bash 
sha256sum iso-name.iso
```
